## Bienvenido

La API de Jelou te permite enviar mensajes a usuarios a través de WhatsApp. Esta API soporta múltiples tipos de mensajes incluyendo texto, imágenes, audio, video, archivos, stickers, ubicación, contactos y elementos interactivos como botones y respuestas rápidas.

## Endpoint Base

Todos los mensajes se envían a través del siguiente endpoint:

```
POST https://api.jelou.ai/v1/bots/{botId}/messages
```

## Requisitos de Sesión

Esta API solo puede enviar mensajes cuando existe una sesión activa: dispones de 24 horas desde el último mensaje del usuario para responder con cualquier tipo de mensaje.

Si la sesión expiró, debes enviar plantillas preaprobadas mediante nuestra plataforma o API de [Campañas](/api-reference/campanas/introduccion) para reabrir la conversación antes de continuar con mensajes libres.

## Campos Comunes

Todos los tipos de mensajes comparten estos campos comunes:

<ParamField body="type" type="string" required>
  Tipo de mensaje a enviar. Ver tipos disponibles en las siguientes páginas.
</ParamField>

<ParamField body="userId" type="string" required>
  Identificador único del usuario que recibirá el mensaje.
</ParamField>

<ParamField body="botId" type="string">
  Identificador único del bot que envía el mensaje. Puede incluirse en el cuerpo o inferirse del parámetro de ruta.
</ParamField>

## Respuestas

### Respuesta Exitosa

```json
{
  "success": true,
  "messageId": "msg_123456789"
}
```

### Respuestas de Error

```json
{
  "error": "INVALID_REQUEST",
  "message": "Formato de mensaje inválido"
}
```

## Códigos de Estado HTTP

- `200` - Éxito
- `400` - Solicitud incorrecta (formato de mensaje inválido o campos requeridos faltantes)
- `401` - No autorizado (sesión inexistente o expirada)
- `404` - No encontrado (bot o usuario no encontrado)
- `500` - Error interno del servidor
