## Descripción

El nodo **Lista Numerada** te permite organizar información en formato de lista numerada. Ideal para instrucciones paso a paso, procesos secuenciales y contenido que requiere orden específico.

## Características principales

- Numeración automática
- Formato ordenado y estructurado
- Ideal para procesos secuenciales
- Fácil de seguir y entender

## Uso básico

Utiliza el nodo Lista Numerada cuando necesites:

- Proporcionar instrucciones paso a paso
- Mostrar procesos secuenciales
- Organizar información en orden específico
- Crear tutoriales o guías

## Configuración

### Propiedades principales

Las propiedades principales del nodo Lista Numerada incluyen:

- **Título**: Encabezado descriptivo de la lista
- **Elementos**: Los elementos numerados de la lista
- **Tipo de numeración**: Números, letras, romanos, etc.
- **Inicio**: Número o letra de inicio (opcional)

## Tipos de numeración

### Numeración estándar

Lista con números arábigos (1, 2, 3...).

```
1. Primer paso
2. Segundo paso
3. Tercer paso
```

### Numeración con letras

Lista con letras (a, b, c... o A, B, C...).

```
a. Opción A
b. Opción B
c. Opción C
```

### Numeración romana

Lista con números romanos (i, ii, iii...).

```
i. Primer elemento
ii. Segundo elemento
iii. Tercer elemento
```

## Ejemplos de uso

### Instrucciones paso a paso

```
Cómo configurar tu cuenta:

1. Ingresa a tu perfil
2. Selecciona "Configuración"
3. Completa tu información
4. Guarda los cambios
```

### Proceso de registro

```
Pasos para registrarte:

1. Completa el formulario
2. Verifica tu correo electrónico
3. Crea tu contraseña
4. Inicia sesión
```

<Tip>
Usa listas numeradas cuando el orden de los elementos sea importante. Para listas sin orden específico, considera usar el [nodo Lista](/nodos/lista) en su lugar.
</Tip>

<Warning>
Mantén las listas numeradas concisas. Si una lista es muy larga (más de 10 elementos), considera dividirla en múltiples secciones o usar otros formatos.
</Warning>

## Mejores prácticas

- Usa títulos descriptivos para las listas
- Mantén cada elemento enfocado en un solo concepto
- Usa lenguaje claro y directo
- Proporciona contexto antes de la lista cuando sea necesario
- Verifica que el orden de los elementos sea lógico
