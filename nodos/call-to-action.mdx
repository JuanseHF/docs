## Descripción

El nodo **Call to Action** (CTA) te permite crear llamadas a la acción destacadas que dirigen a los usuarios hacia acciones específicas e importantes. Es ideal para conversiones y acciones clave.

## Características principales

- Diseño destacado y visualmente atractivo
- Múltiples tipos de acciones
- Personalización de estilo y diseño
- Seguimiento de interacciones

## Uso básico

Utiliza el nodo Call to Action cuando necesites:

- Destacar acciones importantes
- Aumentar conversiones
- Guiar a los usuarios hacia pasos críticos
- Crear puntos de decisión claros

## Configuración

### Propiedades principales

Las propiedades principales del nodo Call to Action incluyen:

- **Texto principal**: Mensaje principal de la llamada a la acción
- **Texto del botón**: Texto del botón de acción
- **Tipo de acción**: Registro, compra, descarga, etc.
- **Destino**: URL o acción a ejecutar
- **Estilo**: Diseño visual del CTA (destacado, primario, secundario)

## Tipos de Call to Action

### CTA de registro

Invita a los usuarios a registrarse.

```
¡Únete a nosotros hoy!
[Botón: Crear cuenta gratis]
```

### CTA de compra

Dirige a los usuarios hacia una compra.

```
Obtén acceso completo ahora
[Botón: Comprar ahora]
```

### CTA de descarga

Invita a descargar un recurso.

```
Descarga nuestra guía gratuita
[Botón: Descargar PDF]
```

### CTA de contacto

Fomenta el contacto con el equipo.

```
¿Tienes preguntas? Estamos aquí para ayudarte
[Botón: Contactar ahora]
```

## Ejemplos de uso

### CTA destacado

```
¡Oferta especial limitada!
Ahorra 50% en tu primer mes
[Botón: Aprovechar oferta]
Estilo: Destacado
```

### CTA con contexto

```
Has completado todos los pasos
Ahora puedes comenzar a usar todas las funciones
[Botón: Comenzar ahora]
```

<Tip>
Usa CTAs de manera estratégica y no excesiva. Demasiados CTAs pueden diluir su efectividad y abrumar al usuario.
</Tip>

<Warning>
Asegúrate de que el CTA sea relevante al contexto de la conversación. Los CTAs fuera de contexto pueden frustrar a los usuarios.
</Warning>

## Mejores prácticas

- Usa lenguaje claro y orientado a la acción
- Crea un sentido de urgencia cuando sea apropiado
- Proporciona valor claro en el mensaje
- Diseña CTAs visualmente destacados pero no intrusivos
- Prueba diferentes textos y estilos para optimizar conversiones
