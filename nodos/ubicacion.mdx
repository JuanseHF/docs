## Configuración

Puedes elegir la ubicación directamente en el mapa de la plataforma o enviarla vía API con longitud y latitud. Consulta la [referencia de API](/api-reference/envia-mensajes/ubicacion) para más detalles sobre el envío mediante API.

Este nodo envía una ubicación de WhatsApp al usuario, que podrá abrir en su aplicación de mapas.

Para solicitar la ubicación del usuario mediante un botón, utiliza el [nodo Solicitud de Ubicación](/nodos/solicitud-ubicacion).
