## Descripción

El nodo **Ubicación** te permite compartir ubicaciones geográficas con tus usuarios. Los usuarios pueden abrir la ubicación en sus aplicaciones de mapas preferidas.

## Características principales

- Coordenadas geográficas precisas
- Direcciones legibles
- Integración con aplicaciones de mapas
- Vista previa del mapa

## Uso básico

Utiliza el nodo Ubicación cuando necesites:

- Compartir direcciones de oficinas o tiendas
- Indicar puntos de encuentro
- Proporcionar ubicaciones de eventos
- Guiar a los usuarios a ubicaciones específicas

## Configuración

### Propiedades principales

Las propiedades principales del nodo Ubicación incluyen:

- **Dirección**: Dirección legible de la ubicación
- **Coordenadas**: Latitud y longitud (opcional)
- **Nombre del lugar**: Nombre descriptivo del lugar
- **Descripción**: Información adicional sobre la ubicación
- **Radio**: Radio de búsqueda (opcional)

## Ejemplos de uso

### Ubicación simple

```
Ubicación: Calle Gran Vía 1, Madrid, España
```

### Ubicación con detalles

```
Nombre: Oficina Principal
Dirección: Calle Gran Vía 1, 28013 Madrid, España
Coordenadas: 40.4168° N, 3.7038° W
Descripción: Nuestra oficina principal está ubicada en el centro de Madrid.
```

<Tip>
Proporciona siempre una dirección legible además de las coordenadas para que los usuarios puedan entender fácilmente la ubicación.
</Tip>

<Warning>
Asegúrate de tener permiso para compartir ubicaciones y que la información sea precisa y actualizada.
</Warning>

## Formatos de coordenadas

### Grados decimales

```
Latitud: 40.4168
Longitud: -3.7038
```

### Grados, minutos y segundos

```
40°25'0.48"N, 3°42'13.68"W
```

## Mejores prácticas

- Proporciona direcciones completas y precisas
- Incluye información de referencia cuando sea útil
- Verifica que las coordenadas sean correctas
- Considera incluir instrucciones de cómo llegar si es necesario
