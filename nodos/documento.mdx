## Descripción

El nodo **Documento** te permite compartir archivos y documentos con tus usuarios. Soporta múltiples formatos de documentos para diferentes necesidades.

## Características principales

- Soporte para múltiples formatos de archivo
- Vista previa de documentos
- Descripción y metadatos
- Descarga de archivos

## Uso básico

Utiliza el nodo Documento cuando necesites:

- Compartir PDFs, documentos de texto o hojas de cálculo
- Enviar archivos de configuración o plantillas
- Proporcionar documentación descargable
- Compartir recursos con tus usuarios

## Configuración

### Propiedades principales

Las propiedades principales del nodo Documento incluyen:

- **Archivo**: El archivo que se compartirá (subido o enlace)
- **Nombre del archivo**: Nombre que verá el usuario
- **Descripción**: Descripción del contenido del documento
- **Tipo de archivo**: Formato del documento
- **Tamaño**: Tamaño del archivo (mostrado al usuario)

## Formatos soportados

### Documentos

- **PDF**: Documentos portátiles
- **DOC/DOCX**: Documentos de Microsoft Word
- **TXT**: Archivos de texto plano
- **RTF**: Formato de texto enriquecido

### Hojas de cálculo

- **XLS/XLSX**: Hojas de cálculo de Excel
- **CSV**: Archivos de valores separados por comas

### Otros formatos

- **ZIP**: Archivos comprimidos
- **JSON**: Archivos de datos estructurados
- **XML**: Archivos de marcado extensible

## Ejemplos de uso

### Documento simple

```
[Documento: Guía de usuario.pdf]
Descripción: Esta guía contiene instrucciones completas para usar nuestro servicio.
```

### Documento con información adicional

```
[Documento: Plantilla de configuración.json]
Descripción: Usa este archivo como plantilla para configurar tu cuenta.
Tamaño: 2.5 MB
```

<Warning>
Asegúrate de que los archivos que compartes no contengan información sensible y cumplan con las políticas de privacidad y seguridad.
</Warning>

<Tip>
Proporciona siempre una descripción clara del contenido del documento para que los usuarios sepan qué esperar antes de descargarlo.
</Tip>

## Mejores prácticas

- Mantén los archivos en un tamaño razonable (idealmente menos de 10 MB)
- Proporciona descripciones claras del contenido
- Verifica que los archivos sean seguros antes de compartirlos
- Considera usar formatos ampliamente compatibles como PDF
