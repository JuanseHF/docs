## Descripción

El nodo **Imagen** te permite compartir imágenes con tus usuarios. Soporta diferentes formatos de imagen y puede incluir descripciones y texto adicional.

## Características principales

- Soporte para múltiples formatos (JPG, PNG, GIF, WebP)
- Imágenes estáticas y animadas
- Texto descriptivo opcional
- Enlaces a imágenes externas o archivos subidos

## Uso básico

Utiliza el nodo Imagen cuando necesites:

- Mostrar productos o servicios visualmente
- Ilustrar instrucciones o tutoriales
- Compartir infografías o gráficos
- Añadir elementos visuales a tus conversaciones

## Configuración

### Propiedades principales

Las propiedades principales del nodo Imagen incluyen:

- **URL de la imagen**: Enlace a la imagen o archivo subido
- **Texto alternativo**: Descripción de la imagen para accesibilidad
- **Pie de foto**: Texto opcional que aparece debajo de la imagen
- **Formato**: Tipo de imagen (estática o animada)

## Ejemplos de uso

### Imagen simple

```
[Imagen: Logo de la empresa]
```

### Imagen con descripción

```
[Imagen: Diagrama del proceso]
Descripción: Este diagrama muestra el flujo completo del proceso de registro.
```

<Warning>
Asegúrate de que las imágenes tengan un tamaño adecuado para evitar tiempos de carga lentos. Se recomienda optimizar las imágenes antes de subirlas.
</Warning>

## Formatos soportados

- **JPG/JPEG**: Para fotografías y imágenes con muchos colores
- **PNG**: Para imágenes con transparencia
- **GIF**: Para imágenes animadas simples
- **WebP**: Formato moderno con mejor compresión

<Tip>
Utiliza imágenes optimizadas para web para mejorar la velocidad de carga y la experiencia del usuario.
</Tip>
