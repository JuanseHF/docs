## Configuración

- **Contenido**: El texto del mensaje que se enviará al usuario

## Usar variables en tus mensajes

Puedes usar cualquier tipo de variables en tus mensajes.

```
Hola {{$user.names}}, el status de tu pedido es: {{$memory.pedido.status}}
```
