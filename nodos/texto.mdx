## Descripción

El nodo **Texto** te permite enviar mensajes de texto simples a tus usuarios. Es el tipo de nodo más básico y versátil para comunicarte con tus usuarios.

## Características principales

- Mensajes de texto personalizables
- Soporte para variables dinámicas
- Formato de texto básico
- Integración con otros nodos

## Uso básico

Utiliza el nodo Texto cuando necesites:

- Enviar mensajes informativos
- Proporcionar instrucciones paso a paso
- Confirmar acciones del usuario
- Personalizar mensajes con variables

## Configuración

### Propiedades principales

Las propiedades principales del nodo Texto incluyen:

- **Contenido del mensaje**: El texto que se enviará al usuario
- **Variables**: Valores dinámicos que se pueden insertar en el mensaje
- **Formato**: Opciones básicas de formato de texto

## Ejemplos de uso

### Mensaje simple

```
Bienvenido a Jelou. ¿En qué puedo ayudarte hoy?
```

### Mensaje con variables

```
Hola {{nombre}}, tu pedido #{{numero_pedido}} está siendo procesado.
```

<Tip>
Utiliza variables para personalizar los mensajes y hacer que cada conversación se sienta única para cada usuario.
</Tip>
