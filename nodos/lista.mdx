## Descripción

El nodo **Lista** te permite mostrar información en formato de lista con opciones que los usuarios pueden seleccionar. Ideal para menús, opciones y navegación estructurada.

## Características principales

- Listas con opciones seleccionables
- Múltiples formatos de lista
- Descripciones opcionales para cada elemento
- Acciones personalizables por elemento

## Uso básico

Utiliza el nodo Lista cuando necesites:

- Mostrar menús de opciones
- Presentar catálogos de productos o servicios
- Crear navegación estructurada
- Ofrecer múltiples opciones al usuario

## Configuración

### Propiedades principales

Las propiedades principales del nodo Lista incluyen:

- **Título de la lista**: Encabezado descriptivo de la lista
- **Elementos**: Los elementos individuales de la lista
- **Descripción**: Descripción opcional de cada elemento
- **Acción**: Acción que se ejecuta al seleccionar un elemento
- **Formato**: Estilo de visualización de la lista

## Tipos de listas

### Lista simple

Lista básica con elementos de texto.

```
- Opción 1
- Opción 2
- Opción 3
```

### Lista con descripciones

Lista donde cada elemento tiene una descripción adicional.

```
• Producto A
  Descripción: Descripción del producto A
  
• Producto B
  Descripción: Descripción del producto B
```

### Lista interactiva

Lista donde cada elemento es seleccionable y ejecuta una acción.

```
[Seleccionar] Opción 1 → Navega a sección 1
[Seleccionar] Opción 2 → Navega a sección 2
```

## Ejemplos de uso

### Menú de opciones

```
Título: ¿En qué puedo ayudarte?
- Información de productos
- Soporte técnico
- Consultas de facturación
- Contactar con ventas
```

### Catálogo de productos

```
Título: Nuestros productos
• Plan Básico - $9.99/mes
• Plan Premium - $19.99/mes
• Plan Empresarial - $49.99/mes
```

<Tip>
Mantén las listas concisas y organizadas. Idealmente, muestra entre 3 y 7 opciones para evitar abrumar al usuario.
</Tip>

<Warning>
Evita crear listas demasiado largas. Si tienes muchas opciones, considera dividirlas en múltiples listas o usar otros tipos de nodos como [botones](/nodos/botones).
</Warning>

## Mejores prácticas

- Usa títulos descriptivos para las listas
- Mantén los elementos de la lista consistentes en formato
- Proporciona descripciones claras cuando sea necesario
- Limita el número de elementos para mejor usabilidad
