## Descripción

El nodo **Contacto** te permite compartir información de contacto de forma estructurada con tus usuarios. Los usuarios pueden guardar fácilmente la información en sus contactos.

## Características principales

- Información de contacto estructurada
- Múltiples métodos de contacto
- Guardado directo en contactos del usuario
- Información de organización opcional

## Uso básico

Utiliza el nodo Contacto cuando necesites:

- Compartir información de contacto de soporte
- Proporcionar detalles de contacto de un equipo
- Facilitar que los usuarios guarden información de contacto
- Compartir tarjetas de contacto digitales

## Configuración

### Propiedades principales

Las propiedades principales del nodo Contacto incluyen:

- **Nombre**: Nombre completo del contacto
- **Teléfono**: Número de teléfono
- **Correo electrónico**: Dirección de correo electrónico
- **Organización**: Nombre de la organización (opcional)
- **Dirección**: Dirección física (opcional)
- **Sitio web**: URL del sitio web (opcional)
- **Notas**: Información adicional (opcional)

## Ejemplos de uso

### Contacto simple

```
Nombre: Juan Pérez
Teléfono: +34 123 456 789
Email: juan@ejemplo.com
```

### Contacto completo

```
Nombre: María García
Organización: Jelou Support
Teléfono: +34 987 654 321
Email: soporte@jelou.com
Sitio web: https://jelou.com
Dirección: Calle Principal 123, Madrid
```

<Tip>
Incluye siempre al menos un método de contacto principal (teléfono o correo electrónico) para que los usuarios puedan comunicarse fácilmente.
</Tip>

<Warning>
Asegúrate de tener permiso para compartir información de contacto y que cumpla con las regulaciones de privacidad aplicables.
</Warning>

## Mejores prácticas

- Proporciona información de contacto verificada y actualizada
- Incluye múltiples métodos de contacto cuando sea posible
- Usa formatos estándar para números de teléfono (con código de país)
- Verifica que las direcciones de correo electrónico sean válidas
