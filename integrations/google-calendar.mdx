## Configuración básica

Puedes instalar la integración de **Google Calendar** directamente desde nuestro Marketplace. Una vez instalada, puedes acceder a Google Calendar y gestionar eventos automáticamente.

### Create events JSON example

```json
{
  "summary": "Onboarding Brain Studio",
  "description": "Sesión de implementación con el cliente ACME.",
  "start": {
    "dateTime": "2025-01-16T14:00:00-03:00"
  },
  "end": {
    "dateTime": "2025-01-16T15:00:00-03:00"
  },
  "attendees": [
    { "email": "cliente@example.com" },
    { "email": "success@example.com" }
  ],
  "conferenceData": {
    "createRequest": {
      "requestId": "brain-studio-onboarding",
      "conferenceSolutionKey": { "type": "hangoutsMeet" }
    }
  },
  "reminders": {
    "useDefault": false,
    "overrides": [
      { "method": "email", "minutes": 24 * 60 },
      { "method": "popup", "minutes": 10 }
    ]
  }
}
```

> Reemplaza los valores de `dateTime`, asistentes y recordatorios según las respuestas de tus usuarios dentro del flujo.
